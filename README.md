# Classic Asteroids

A browser-based classic Asteroids game built with plain HTML/CSS/JavaScript.

## Run

From this folder:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Controls

- Turn: Left/Right arrows or `A`/`D`
- Thrust: Up arrow or `W`
- Shoot: `Space`
- Start/Restart: `Enter`
